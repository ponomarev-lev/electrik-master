Минимальное требование:
Установлен Python 3.

Запуск на Windows:
Дважды кликнуть start.bat

Остановка на Windows:
Дважды кликнуть stop.bat

Запуск на macOS:
Дважды кликнуть start.command

Остановка на macOS:
Дважды кликнуть stop.command

Если macOS не дает запустить .command:
Открыть Terminal в папке проекта и один раз выполнить:
chmod +x start.command stop.command

Ручной запуск через Python:
python launcher.py start
python launcher.py stop
python launcher.py status
